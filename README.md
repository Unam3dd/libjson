# 🧠 JSON Parser in C

A fast, lightweight JSON parser written in pure C.  
Combines a finite state machine (FSM) lexer with an abstract syntax tree (AST) builder.

---

## 🚀 Features

- ⚙️ FSM-based lexical analysis  
- 🌲 Full JSON parsing with AST construction  
- ⚡ Zero external dependencies  
- 💡 Clean C89/C99-compatible code  
- 🧪 Designed for speed and embedded use
